# Інструкції для адміністратора

## Як зайти в адмін-панель

### 1. Вхід в систему
- Перейдіть на головну сторінку сайту
- Натисніть кнопку **"Вхід"** у верхньому правому куті
- Введіть облікові дані адміністратора:
  - **Email:** `admin@fooddelivery.com`
  - **Пароль:** `Admin123!`

### 2. Доступ до адмін-панелі
- Після успішного входу в системі, у меню навігації з'явиться пункт **"Адмін-панель"**
- Натисніть на **"Адмін-панель"** для переходу до панелі управління

Або перейдіть безпосередньо за адресою: `/Admin/Index`

## Функціонал адмін-панелі

### Управління товарами
- **Перегляд товарів:** `/Admin/Products`
- **Додати товар:** Натисніть кнопку "Додати товар" на сторінці управління товарами
- **Редагувати товар:** Натисніть кнопку "Редагувати" біля потрібного товару
- **Видалити товар:** Натисніть кнопку "Видалити" біля потрібного товару (підтвердіть видалення)

### Управління замовленнями
- **Перегляд замовлень:** `/Admin/Orders`
- **Оновлення статусу:** Використовуйте випадаючий список для зміни статусу замовлення

### Управління користувачами та призначення адміністраторів
- **Перегляд користувачів:** `/Admin/Users`
- **Призначити адміністратора:**
  1. Перейдіть до сторінки "Управління користувачами"
  2. Знайдіть користувача, якому потрібно надати права адміністратора
  3. Натисніть кнопку **"Зробити адміном"** біля потрібного користувача
  4. Підтвердіть дію
- **Забрати права адміністратора:**
  1. Перейдіть до сторінки "Управління користувачами"
  2. Знайдіть адміністратора, у якого потрібно забрати права
  3. Натисніть кнопку **"Забрати права адміна"** біля потрібного користувача
  4. Підтвердіть дію

## Пошук та фільтрація товарів

### На головній сторінці доступні:
- **Пошук товарів:** Введіть назву або опис товару в поле пошуку
- **Фільтр по категоріях:** 
  - Використовуйте кнопки категорій для фільтрації
  - Або виберіть категорію з випадаючого списку
- **Комбінований пошук:** Можна одночасно використовувати пошук та фільтр по категорії

## Важливо!
- Роль "Admin" автоматично створюється при першому запуску додатку
- Перший адміністратор створюється автоматично з обліковими даними вище
- Всі адміністратори мають повний доступ до всіх функцій адмін-панелі
